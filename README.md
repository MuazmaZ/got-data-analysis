# got-data-analysis
Game Of Thrones data analysis around the episodes, writers, viewers, and ratings data.

Exploratory Analysis for correspondance between viewers and ratings as well as how writers/directors effect the ratings

# Kaggle Analysis
source: https://www.kaggle.com/muazmaz/analysis-game-of-thrones-episode-data/

# About this dataset
Source Kaggle(https://www.kaggle.com/dasbootstrapping/game-of-thrones-episode-data) and I added Season 8 data 
Game of Thrones is one of the world's most popular TV shows. Which season is the most popular? Is there are link between ratings and viewer figures? Do certain writers and directors make better episodes?
